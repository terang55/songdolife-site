# 송도 이사 완벽 체크리스트

## 📋 이사 3개월 전 준비사항

### 🏠 집 구하기
- [ ] 송도 아파트 단지별 특성 파악
- [ ] 예산 대비 적정 매물 선정
- [ ] 교통 접근성 확인 (지하철, 버스)
- [ ] 주변 편의시설 확인 (마트, 병원, 학교)
- [ ] 관리비 및 유지비용 계산

### 💰 자금 계획
- [ ] 이사비용 예산 수립
- [ ] 전세자금대출 또는 주택담보대출 신청
- [ ] 부동산 중개수수료 준비
- [ ] 보증금 및 월세 준비

### 📄 필요 서류 준비
- [ ] 주민등록등본 (최근 3개월 이내)
- [ ] 소득증명서 (재직증명서, 급여명세서)
- [ ] 인감증명서
- [ ] 통장 사본 (잔고 증명)

## 📅 이사 1개월 전 준비사항

### 🎯 이사업체 선정
- [ ] 3-4개 업체 견적 비교
- [ ] 이사 날짜 예약 (평일이 더 저렴)
- [ ] 포장재 및 박스 준비
- [ ] 특수 물품 (피아노, 냉장고 등) 이동 상담

### 🏢 직장 관련 업무
- [ ] 인사팀에 주소 변경 신고
- [ ] 통근 경로 및 시간 재계산
- [ ] 주차 공간 확보 (회사 및 집)

### 🏫 자녀 교육 관련
- [ ] 전학 신청 (현재 학교)
- [ ] 송도 지역 학교 정보 수집
- [ ] 학원 및 교습소 정보 확인
- [ ] 국제학교 입학 상담 (필요시)

## 🚚 이사 1주일 전 준비사항

### 📞 각종 업체 연락
- [ ] 전기: 한국전력공사 (123번)
- [ ] 가스: 인천도시가스 (032-440-0700)
- [ ] 수도: 인천상수도사업본부 (032-440-3300)
- [ ] 인터넷: KT, SK, LG U+ 등
- [ ] 케이블TV: 각 업체별 연락

### 📦 이사 짐 정리
- [ ] 중요 서류 별도 보관
- [ ] 귀중품 직접 운반 준비
- [ ] 냉장고/김치냉장고 정리 및 전원 차단
- [ ] 세탁기 물빼기
- [ ] 불필요한 물건 정리 (기부 또는 폐기)

## 🏡 이사 당일 체크리스트

### 🌅 이사 시작 전
- [ ] 날씨 확인 (우천 시 대비책)
- [ ] 이사업체 연락 (시간 확인)
- [ ] 귀중품 및 중요 서류 별도 보관
- [ ] 구 집 상태 사진 촬영
- [ ] 현금 준비 (팁 및 급여 지급용)

### 🚛 이사 진행 중
- [ ] 짐 목록 체크 (분실물 확인)
- [ ] 가전제품 포장 상태 확인
- [ ] 박스 번호 및 내용물 확인
- [ ] 이사업체 직원과 소통 유지

### 🏠 새 집 입주 시
- [ ] 집 상태 점검 (하자 여부 확인)
- [ ] 전기, 가스, 수도 개통 확인
- [ ] 엘리베이터 사용 시간 확인
- [ ] 주차 공간 확보
- [ ] 짐 배치 및 정리

## 📋 이사 후 2주 이내 해야 할 일

### 🏛️ 관공서 업무
- [ ] 주민등록 이전 신고 (이사 후 14일 이내)
- [ ] 건강보험 주소 변경
- [ ] 국민연금 주소 변경
- [ ] 자동차 등록 주소 변경
- [ ] 운전면허 주소 변경

### 🏦 금융 관련
- [ ] 은행 주소 변경 (모든 거래 은행)
- [ ] 신용카드 주소 변경
- [ ] 보험 주소 변경 (자동차, 건강, 생명보험)
- [ ] 증권계좌 주소 변경

### 📱 통신 관련
- [ ] 휴대폰 주소 변경
- [ ] 인터넷 설치 및 설정
- [ ] 케이블TV 설치
- [ ] 홈시큐리티 설치 (필요시)

## 🎯 송도 생활 적응 체크리스트

### 🛒 생활 인프라 파악
- [ ] 가까운 대형마트 위치 확인
  - 이마트 송도점, 홈플러스 송도점
- [ ] 24시간 편의점 위치 확인
- [ ] 약국 및 병원 위치 확인
- [ ] 우체국 및 택배 픽업 장소 확인

### 🚇 교통 시스템 익히기
- [ ] 지하철 노선 및 시간표 확인
- [ ] 버스 노선 및 정류장 위치 확인
- [ ] 택시 호출 방법 숙지
- [ ] 교통카드 충전소 위치 확인

### 🏥 응급 상황 대비
- [ ] 가까운 종합병원 위치 확인
  - 가천대 길병원, 인천성모병원
- [ ] 응급실 위치 및 연락처 저장
- [ ] 단지 내 관리사무소 연락처 저장
- [ ] 경찰서 및 소방서 연락처 저장

## 👨‍👩‍👧‍👦 가족별 맞춤 체크리스트

### 👶 영유아 가족
- [ ] 소아과 병원 위치 확인
- [ ] 어린이집 및 유치원 정보 수집
- [ ] 놀이터 및 키즈카페 위치 확인
- [ ] 육아용품 구매처 확인

### 🎓 학생 가족
- [ ] 학교 전학 절차 완료
- [ ] 통학로 및 교통편 확인
- [ ] 학원가 위치 및 정보 수집
- [ ] 도서관 및 문화센터 위치 확인

### 👴 어르신 가족
- [ ] 노인복지관 위치 및 프로그램 확인
- [ ] 경로당 위치 확인
- [ ] 대중교통 할인 혜택 확인
- [ ] 의료진 및 병원 정보 수집

## 💡 송도 생활 꿀팁

### 🎁 혜택 및 할인 정보
- [ ] 송도 주민 할인 혜택 확인
- [ ] 문화센터 프로그램 등록
- [ ] 도서관 회원 가입
- [ ] 체육시설 이용 신청

### 🌟 커뮤니티 참여
- [ ] 단지 내 커뮤니티 가입
- [ ] 송도 주민 온라인 카페 가입
- [ ] 동호회 및 모임 참여
- [ ] 이웃 인사 및 관계 형성
